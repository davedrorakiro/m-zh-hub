# m-zh-hub

## 项目简介

m-zh-hub 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为一系列独立前端页面的集中存放与版本管理之用。

所有页面均为静态 HTML 文件，可直接通过浏览器访问，适用于学习、参考、展示或轻量部署场景。

## 目录说明

```
m-zh-hub/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 公共资源（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立的 HTML 页面。
- `assets/` 目录存放页面间共享的资源文件，按需引用。

## 页面归档说明

- 每个页面均为完整独立的 HTML 文件，包含其所需的内联或外部资源。
- 页面之间无功能依赖，可以单独部署或查看。
- 仓库会定期更新，新增或修正页面内容。
- 页面命名以内容主题或功能描述为主，便于识别。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
3. 也可将 `pages/` 目录部署到任意静态服务器上。

## 维护说明

- 本仓库由维护者不定期更新。
- 欢迎提交 Issue 或 Pull Request 来建议新增页面或修正现有内容。
- 所有提交的页面应保持独立、完整，不包含外部敏感链接或恶意代码。
